# 加载项目上下文

在开始工作前运行此命令，快速了解项目当前状态。

## 步骤

// turbo
1. 运行上下文加载脚本：
```bash
bash .agent/hooks/session-start.sh
```

2. 阅读输出结果，重点关注：
   - **当前分支** — 确认你在正确的分支上
   - **最近提交** — 了解最近做了什么
   - **活跃冲刺** — 当前冲刺目标是什么
   - **Bug 数量** — 是否有未解决的 Bug
   - **代码健康** — TODO/FIXME 数量是否异常

3. 如果检测到**活跃会话状态文件**（`production/session-state/active.md`），读取它以恢复上次工作的上下文。

4. 根据上下文信息，向用户汇报项目当前状态，并建议下一步行动。
