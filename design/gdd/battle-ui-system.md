# 战斗 UI 系统 (Battle UI System)

> **状态**: 已设计
> **最后更新**: 2026-03-25
> **服务支柱**: 视觉轰炸

## 概述

战斗中的所有 HUD 元素：血条、怒气槽、回合指示器、计时器、连击计数、KO 特写文字。UI 必须华丽但不遮挡战斗视野。

## 详细设计

### UI 组件

| 组件 | 位置 | 说明 |
|------|------|------|
| **血条** | 顶部左右 | 渐变色（绿→黄→红），受伤时有"二段掉血"效果 |
| **怒气槽** | 血条下方 | 蓝紫色渐变，满时闪烁金光 |
| **角色头像** | 血条旁 | 圆形头像，低血时切换紧张表情 |
| **角色名字** | 头像下方 | 角色中文名 + 元素图标 |
| **回合标记** | 居中 | ●=赢 ○=未打 |
| **计时器** | 顶部中央 | 大号数字，< 10 秒时红色闪烁 |
| **连击计数** | 命中方向 | "X HITS!" 弹出文字，带缩放动画 |
| **KO 文字** | 屏幕中央 | KO 时大号文字 + 震动 + 慢动作 |
| **ROUND 文字** | 屏幕中央 | 回合开始时显示 "ROUND 1 / FIGHT!" |

### 血条动画

```
受伤时:
  1. 实际血条立即掉到目标值（绿/黄/红色）
  2. "幽灵血条"延迟 0.5 秒后缓慢掉落（白色）
  3. 血量差值 = 本次伤害可视化
  
颜色:
  > 60% → 绿色渐变
  30-60% → 黄色渐变
  < 30% → 红色渐变 + 脉冲闪烁
```

### 连击计数动画

```
命中时:
  1. 数字弹出（scale: 0 → 1.5 → 1.0，duration: 200ms）
  2. 每次 +1 时文字晃动
  3. 5+ hits: 文字变金色
  4. 10+ hits: 文字变红色 + 火焰特效
  5. 连击结束后淡出
```

### 与其他系统交互

| 系统 | 数据流向 | 说明 |
|------|----------|------|
| 角色状态 → 本系统 | 输入 | HP 变化 |
| 怒气系统 → 本系统 | 输入 | 怒气值变化 |
| 连招系统 → 本系统 | 输入 | 连击数 |
| 对战模式 → 本系统 | 输入 | 回合信息、计时 |

## 验收标准

- [ ] 血条二段掉血动画流畅
- [ ] 怒气槽满时视觉反馈明显
- [ ] 连击计数弹出动画华丽
- [ ] 计时器 < 10 秒时红色闪烁
- [ ] UI 不遮挡战斗区域
- [ ] KO / ROUND 文字演出震撼
