# 对战模式管理 (Battle Mode Manager)

> **状态**: 已设计
> **最后更新**: 2026-03-25
> **服务支柱**: 上手即爽

## 概述

管理完整的对战流程：角色选择 → 场景加载 → 回合制对战 → 胜负判定 → 结算。支持本地双人对战（MVP）和后续在线对战扩展。

## 详细设计

### 对战流程

```
[角色选择] → [场景加载] → [Round Start] → [战斗中] → [Round End]
                              ↑                          ↓
                              └──── [下一回合] ←── [判定]
                                                    ↓ (有人赢2局)
                                               [Match End]
                                                    ↓
                                              [胜利/失败画面]
                                                    ↓
                                           [返回角色选择 / 重赛]
```

### 回合制规则

```
总回合数: Best of 3（先赢 2 局）
回合时间: 60 秒
时间归零: 血量高者获胜；血量相同则平局（附加回合）
最大回合: 5（含附加回合）

回合开始:
  1. 显示 "ROUND X" 文字（1.5秒）
  2. 显示 "FIGHT!" 文字（0.5秒）
  3. 双方恢复满血、位置重置
  4. 开始战斗

回合结束:
  1. 一方 HP ≤ 0 → 触发 KO 演出（2秒）
  2. 显示 "KO!" 文字
  3. 胜利方播放胜利动画
  4. 更新比分
```

### Match 结算

```
胜利条件: 先赢 2 局
显示: 胜利角色站姿 + "YOU WIN!" / "VICTORY!"
选项:
  - 重赛（角色不变）
  - 返回角色选择
  - 返回主菜单
```

### 本地双人

```
输入分区:
  P1: 键盘左区 (WASD + JKL)
  P2: 键盘右区 (方向键 + 小键盘)
  
屏幕布局:
  P1 血条/怒气 ← 左上
  P2 血条/怒气 → 右上
  回合计时 → 顶部中央
  连击计数 → 命中位置弹出
```

### 与其他系统交互

| 系统 | 数据流向 | 说明 |
|------|----------|------|
| 菜单 UI → 本系统 | 输入 | 进入对战模式 |
| 角色数据 → 本系统 | 查询 | 加载角色 |
| 场景系统 → 本系统 | 查询 | 加载场景 |
| 战斗系统 ↔ 本系统 | 双向 | 战斗逻辑和胜负判定 |
| 本系统 → 战斗 UI | 输出 | 回合信息、计时 |

## 调优参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `ROUND_TIME` | 60 | 回合时间（秒） |
| `ROUNDS_TO_WIN` | 2 | 胜利所需回合数 |
| `MAX_ROUNDS` | 5 | 最大回合数 |
| `ROUND_START_DELAY` | 2.0 | "ROUND X" 显示时间 |
| `KO_DELAY` | 2.0 | KO 演出时间 |
