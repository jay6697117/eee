# 全女格斗 · 异时空格斗女神

> **Game Concept Document v1.0**
> 创建日期：2026-03-25

## 1. 电梯演讲

来自不同时代和世界的 8 位传奇女性格斗家，各自掌握独特的元素异能，在时空竞技场中以快节奏连招对决，争夺"格斗女神"称号。简化操作、华丽特效、策略性元素克制 — 一款致敬拳皇的全女性格斗游戏。

## 3. 核心循环

### 3.1 30 秒循环（逐帧体验）

玩家通过**简化指令**操作角色：
- **轻攻击 / 重攻击**：方向键 + 攻击键的简单组合
- **必杀技**：一键必杀 + 方向组合（↓→+攻击 等简单指令）
- **元素爆发**：特殊按键释放元素技能，触发视觉特效

核心满足感来源：
- 连招命中时的**打击停顿 (Hit Stop)** 和**屏幕震动**
- 元素特效的**粒子爆炸**（火焰燃烧、冰霜扩散、雷电闪烁）
- 成功连接必杀技时的**慢动作特写**

### 3.2 5 分钟循环（单局对战）

一局对战 = 2-3 回合：
- **开局**：试探对手属性，制定元素克制策略
- **中期**：积攒怒气槽，寻找连招机会
- **高潮**：触发 MAX 觉醒模式，释放超必杀技
- **收尾**：觉醒状态下的决胜一击

"再来一局"心理驱动：快速匹配 + 角色多样性 + 想尝试不同元素组合

### 3.3 单次游玩循环（30-60 分钟）

- 对战模式：连续匹配多局，尝试不同角色
- 训练模式：练习连招和元素互动组合
- 自然停点：每 3-5 局后的战绩结算

### 3.4 长期循环（数周）

- 解锁全部角色的连招列表
- 掌握元素克制和互动组合
- 挑战更高难度的 AI 或更强的玩家
- DLC 新角色带来的新鲜感

## 4. 战斗系统

### 4.1 操作设计（简化型）

```
基础操作：
├── 移动：← → (左右移动)
├── 跳跃：↑
├── 蹲下：↓
├── 轻攻击：A 键
├── 重攻击：B 键
├── 元素技能：C 键
├── 必杀技：→ + A+B / ↓→ + C（简单方向组合）
└── 超必杀技：怒气满时 A+B+C
```

设计原则：**30 秒上手，一辈子精通**
- 新手按按钮就能打出好看的招式
- 高手通过时机、间距、元素搭配提升深度

### 4.2 觉醒系统（怒气槽）

- 攻击命中、被命中、防御成功都会积攒怒气
- 怒气槽满后可手动触发 **MAX 觉醒模式**（限时 10 秒）
- 觉醒状态：攻击力 +30%、速度 +20%、解锁超必杀技
- 觉醒期间元素技能效果增强（火焰面积更大、冰冻时间更长等）

### 4.3 元素系统

#### 元素克制（额外 20% 伤害）

```
🔥 火 → ❄️ 冰 → ⚡ 雷 → 🌊 水 → 🔥 火
🌪️ 风 → 🪨 地 → ⚡ 雷
✨ 光 ↔ 🌑 暗（互相克制）
```

#### 元素互动（环境效果）

当特定元素技能在同一区域碰撞时，触发环境效应：

| 组合 | 效果 | 描述 |
|------|------|------|
| 🔥 + 🌪️ | **火焰风暴** | 场地中央出现持续伤害旋涡 |
| ❄️ + ⚡ | **冰冻闪电** | 地面结冰 + 落雷，减速并麻痹 |
| 🌊 + 🪨 | **泥石流** | 场地一侧被泥浆覆盖，减速区域 |
| ✨ + 🌑 | **混沌爆发** | 全屏闪光 + 双方短暂眩晕 |
| 🔥 + 🌊 | **蒸汽迷雾** | 视野遮蔽，近距离才能看到对手 |
| 🌪️ + ⚡ | **雷暴** | 随机落雷，无差别攻击 |

## 5. 角色阵容（8 名基础角色）

### 🔥 伊格尼斯 (Ignis) — 罗马角斗士
- **元素**：火
- **战斗风格**：近距离压制型，高伤害连招
- **设定**：古罗马最强女角斗士，以不败战绩闻名竞技场
- **必杀技**：烈焰角斗（全身燃烧冲刺突进 + 上挑连击）
- **超必杀**：火山爆发（地面裂开岩浆喷射）

### ❄️ 凌霜 (Ling Shuang) — 中国功夫女侠
- **元素**：冰
- **战斗风格**：反击型，精准防守 + 冰冻控制
- **设定**：武当山冰玉宫传人，以柔克刚
- **必杀技**：寒冰掌（防御成功后反击冻结对手）
- **超必杀**：千里冰封（全场降温，对手移动速度大幅下降 + 冰柱突击）

### ⚡ 奈欧 (Neo) — 赛博朋克黑客
- **元素**：雷
- **战斗风格**：速攻型，高速移动 + 多段轻攻击
- **设定**：2099 年地下网络的传奇黑客，植入战斗芯片后成为格斗天才
- **必杀技**：电磁脉冲（范围麻痹 + 瞬移后突袭）
- **超必杀**：系统过载（召唤数据风暴，全屏雷电轰击）

### 🌊 玛琳 (Marine) — 加勒比女海盗
- **元素**：水
- **战斗风格**：中距离控制型，水流鞭打 + 推拉操控
- **设定**：17 世纪最令人畏惧的女海盗船长，传说能操控海洋
- **必杀技**：海妖之歌（水柱旋涡困住对手）
- **超必杀**：巨浪吞噬（召唤海啸横扫全场）

### 🌪️ 樱 (Sakura) — 日本武士少女
- **元素**：风
- **战斗风格**：居合斩型，蓄力 + 爆发瞬斩
- **设定**：战国时代最年轻的女剑豪，刀法快如疾风
- **必杀技**：神风居合（瞬移拔刀斩，命中后产生真空刃）
- **超必杀**：天照风神斩（连续 7 刀居合，最后一击斩出巨大风刃）

### 🪨 盖亚 (Gaia) — 亚马逊战士
- **元素**：地
- **战斗风格**：重击型，超级铠甲 + 投技
- **设定**：亚马逊雨林深处部落的最强女战士，与大地共鸣
- **必杀技**：大地震击（跺地产生震波 + 石柱突刺）
- **超必杀**：盖亚之怒（召唤石巨人一起攻击）

### ✨ 赛勒涅 (Selene) — 古希腊月之女神
- **元素**：光
- **战斗风格**：远程型，光束投射 + 圣盾防御
- **设定**：奥林匹斯山的月光守护者，以神圣之力净化一切
- **必杀技**：月光箭（远程追踪光弹）
- **超必杀**：圣光审判（从上方降下巨大光柱，全屏判定）

### 🌑 莉莉丝 (Lilith) — 暗夜女王
- **元素**：暗
- **战斗风格**：诡计型，影分身 + 吸血 + 瞬移
- **设定**：被遗忘的暗夜王国末代女王，操控影子进行战斗
- **必杀技**：暗影缠绕（影子触手束缚 + 吸血回复）
- **超必杀**：永夜降临（全场变暗，只有莉莉丝能看到对手，连续暗影突袭）

## 6. 游戏模式

### 6.1 对战模式
- **本地对战**：同一设备双人对战（键盘分区 / 双手柄）
- **在线对战**：匹配对战 + 好友房间
- **排位系统**（后期）：段位制 + 赛季结算

### 6.2 训练模式
- **自由训练**：无限血量 / 无限怒气，练习任意连招
- **连招挑战**：每个角色 10 个预设连招目标
- **元素实验室**：测试元素克制和互动效果

## 7. 游戏支柱 (Design Pillars)

### 支柱 1：上手即爽 (Instant Gratification)
> 任何人拿起手机 30 秒内就能打出华丽的招式和连击。

**设计测试**：如果在"简化操作"和"增加操作深度"之间纠结，选简化。

### 支柱 2：视觉轰炸 (Visual Spectacle)
> 每一拳、每一招都必须让屏幕炸裂，元素特效要让玩家想截图分享。

**设计测试**：如果一个技能打击感不够炸裂，就重做它的特效，不要妥协。

### 支柱 3：个性鲜明 (Distinct Identity)
> 8 个角色不只是数值差异，她们的战斗方式、视觉风格、音效、场景必须让人一眼区分。

**设计测试**：如果两个角色玩起来感觉很像，必须重新设计其中一个。

### 支柱 4：策略深度 (Strategic Depth)
> 元素克制和互动系统让选人和战斗都充满策略思考。

**设计测试**：如果元素系统变成了"无脑选克制属性就赢"，需要调整平衡。

### 反支柱（这个游戏不做的事）

- ❌ **不做复杂搓招**：不会要求玩家记住 ↓↘→↓↘→ + 重拳 这种指令
- ❌ **不做写实画风**：永远保持动漫赛璐璐风格
- ❌ **不做重度剧情模式**：故事通过角色背景和对战前对话呈现，不做 RPG 式长剧情
- ❌ **不做付费变强**：DLC 角色可以有，但不能出现付费数值碾压

## 8. 目标玩家分析

### 主要玩家类型：竞争者 (Competitors)
- 追求技术提升和排名上升
- 喜欢研究角色对策和元素搭配
- 会反复练习连招直到完美

### 次要吸引：成就者 (Achievers) + 表达者 (Expressors)
- 想解锁所有角色、掌握所有连招
- 喜欢用不同角色展现个人风格

### 不适合的玩家
- 追求深度叙事体验的玩家
- 讨厌 PvP 竞技的玩家
- 偏好开放世界探索的玩家

### 市场参考
| 同类游戏 | 借鉴点 |
|----------|--------|
| **拳皇 (KOF)** | 经典格斗框架、连招系统 |
| **Skullgirls** | 全女阵容、独特角色设计 |
| **碧蓝幻想 Versus** | 简化操作、一键必杀 |
| **Brawlhalla** | 网页/多平台、免费模式 |

## 9. 可行性与技术方案

### 9.1 引擎推荐

**推荐方案：网页技术栈（HTML5 Canvas / WebGL）**

理由：
- ✅ 天然支持网页 + PC + 手机三端
- ✅ 独立开发者更易掌控
- ✅ 部署简单（一个 URL 即可访问）
- ✅ 用户历史项目是 Vite + React + Three.js，技术栈匹配

具体技术栈建议：
- **渲染**：PixiJS（2D 渲染性能优秀）或 Phaser（游戏框架丰富）
- **构建**：Vite
- **状态管理**：Zustand（轻量级）
- **网络对战**：WebSocket + WebRTC（后期）

### 9.2 美术管线

- AI 辅助生成角色立绘和场景（Google Nano Banana）
- 动漫赛璐璐风格适合 AI 生成后人工微调
- 战斗动画：帧动画（Sprite Sheet）
- 预估每个角色需要：站立、移动、跳跃、蹲、轻攻击、重攻击、元素技能、必杀技、超必杀、受伤、倒地 = 约 11 组动画

### 9.3 内容规模

| 内容 | 数量 |
|------|------|
| 基础角色 | 8 |
| DLC 角色 | 2-4（后期）|
| 战斗场景 | 8（每角色一个专属场景）|
| BGM | 8 + 1 主菜单 + 1 选人 = 10 首 |
| 游戏模式 | 2（对战 + 训练）|

### 9.4 MVP 定义（最小可玩版本）

**MVP = 2 个角色 + 1 个场景 + 本地对战**

测试核心问题："简化操作的格斗游戏在网页端是否有足够的打击感和乐趣？"

### 9.5 最大风险

| 风险 | 严重度 | 应对 |
|------|--------|------|
| 网页端打击感不足 | 🔴 高 | 优先实现 Hit Stop、屏幕震动、粒子特效 |
| 角色动画工作量大 | 🟡 中 | AI 生成 + 帧数精简（每动作 4-8 帧）|
| 元素平衡难度 | 🟡 中 | 先做纯视觉，MVP 验证后再加数值克制 |
| 手机端操作体验 | 🟡 中 | 虚拟摇杆 + 大按钮设计 |

### 9.6 范围分层

| 层级 | 内容 |
|------|------|
| **Tier 1（必须）** | 2 角色、本地对战、训练模式、1 场景 |
| **Tier 2（核心）** | 8 角色、8 场景、元素系统、怒气觉醒 |
| **Tier 3（完整）** | 在线对战、排位系统、元素互动效果 |
| **Tier 4（扩展）** | DLC 角色、故事模式碎片、赛季内容 |

## 10. 玩家动机分析（自我决定理论）

- **自主性 (Autonomy)**：选择角色 + 选择元素搭配策略 + 自由选择连招路线
- **胜任感 (Competence)**：从按键乱按到精准连招的成长弧线，训练模式提供明确的技能阶梯
- **关联性 (Relatedness)**：每个角色都有独特的性格和背景故事，与对手的战前对话建立角色关系

## 11. 心流设计 (Flow State)

```
技能 ↑
  │        ┌─────────┐
  │   焦虑 │         │
  │        │  心流   │ ← 目标区域
  │        │  通道   │
  │        │         │
  │   无聊 └─────────┘
  └──────────────────→ 挑战 ↑
```

- **新手**：简化操作让新手也能打出华丽招式（避免焦虑）
- **进阶**：元素克制和连招优化提供持续挑战（避免无聊）
- **高手**：觉醒系统时机 + 元素互动 + 间距控制 = 无限深度
